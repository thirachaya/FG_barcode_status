📦 FG Barcode Status Dashboard

A web dashboard for monitoring Finished Goods (FG) barcode status and aging information.

This application visualizes FG data such as totals, aging status, and monthly trends to help users quickly understand inventory conditions and identify potential issues.

Built with React + Vite + TailwindCSS, the dashboard provides a fast and responsive interface for viewing FG status data.

---

✨ Features

- 📊 Dashboard overview
  
  - Display total FG status information
  - Show aggregated inventory metrics

- 📅 Monthly visualization
  
  - Monthly trend chart for FG data

- 📋 Detailed data tables
  
  - Aging table for tracking product aging
  - Detailed table view of FG records

- 📈 Summary metrics
  
  - Total FG count
  - Maximum aging and non-GI indicators

---

🖥 Tech Stack

Frontend

- React
- Vite
- TailwindCSS

⚙️ Installation

1️⃣ Clone the repository

git clone https://github.com/thirachaya/FG_barcode_status.git

2️⃣ Navigate to the project folder

cd FG_barcode_status

3️⃣ Install dependencies

npm install

4️⃣ Run development server

npm run dev

Then open in browser:

http://localhost:5173
